ScalemonsDex V2.3

Ouvre scalemondex_v2_3_standalone.html dans Chrome pour tester sans serveur.

Nouveautés V2.3 :
- le champ Attaque fonctionne avec des bulles : tape une attaque puis Entrée ;
- clique sur une bulle pour retirer l’attaque ;
- plusieurs attaques sont combinées en mode ET : Knock Off + Swords Dance = uniquement les Pokémon qui apprennent les deux ;
- la colonne Attaques NatDex reste supprimée.

La version index.html/style.css/app.js servira pour GitHub Pages. Les talents et attaques National Dex sont chargés depuis les fichiers publics de Pokémon Showdown.

Correctif V2.3 : saisie des champs stabilisée, le champ Attaque ne relance plus le tableau à chaque frappe.
